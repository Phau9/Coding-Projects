Html Elements, Html Attributes and Html Tags

Tags are used as requests to the publisher (web browser) to display the information in a particular way, there are always opening (start) tags and closing (end) tags which surround the contents.

Within the Html Elements there are Attributes, these are traits which are added to the Hmtl Element. such as how to align the text, how large or small the font is, the colour and so on. you can find out more from the devdocs website.

There are tags which do not require a closing tag, which is known as a self-closing tag, such as the BR tag (Line break which gives a space between content).

To type comments inside HMTL which are not shown on the webpage you need to use the following : open the tag (symbol)!-- and close using -->.

HEAD tag requires an opening and a closing (head) (/head). this is the part of the HTML file that holds information about the webpage and how the browser how to handle the page. This can be things like what the webpage is called.

Tables

Within Tables there are several key elements, first being the TR, this refers to the table Row (in excel this would be row 1).

The next important element is the TD, this refers to the table data (cell A in excel or cell B, if you havve multiple TD elements within one TR)

Tables can also contain head/body/foot like a normal html page.

  1. thead is the table header, what you would name your columns
    • to list column A,B,C you would have TH[A column title]/TH then followed by another TH[B column title]/TH and so on
  2. tbody is the table content, i.e your cell data
  3. tfoot is a row that appears at the bottom off the table, however this is barely used as you can use the TR/TD to make a bottom row
  4. Attributes - it is vital to know that table Attributes are no longer used as this would be set up in CSS
    • Border="Number" will give your table an old fashioned border
    • cellspacing="%value or number" will increase the spacing between cells
    • cellpadding="number" puts the space around your content and the border
